发卡平台支付渠道搭建攻略

· 2023-04-30 ·

首先是整体步骤

  1. 先获得营业执照，代办或者自己办理。如果没有也能搞，但会很麻烦。
  2. 通过营业执照开通云闪付收银台，并通过云闪付收银台开通微信收款和支付宝收款
  3. 搭建易支付平台，对接云闪付收银台上的微信和支付宝渠道
  4. 独角卡对接易支付

1. 营业执照 - 个体工商户

1.1. 淘宝代办

优势：快捷，方便
劣势：如果不在当地办理注销麻烦

入口：
淘宝搜索"个体工商户"

1.2. 自己办理

自己去当地的行政服务中心办理即可。
注意点：

  1. 核名，起名字需要注意，最好挑一些正向且冷僻的名字，名字撞车是不允许注册的。
  2. 地址，个体工商户的地址一般只能放在商业地产(产权/租约)或者身份证所属的地址上，这点需要注意，如果两者都不具备，需要考虑代办。

2. 税务登记和年报

  1. 拿到执照后的一个月内记得去做税务登记。
  2. 每年都需要做年报，如果没有业务发生，选择零申报即可。

2. 通过营业执照开通云闪付收银台

2.1. 材料

中国银联普通商户进件都需要材料，根据店铺类型的不同分为四种情况：

第一、个人无营业执照且无固定门店：比如摆地摊的 （小微商户）
需要准备材料：个人身份证（正反面照片）、银行卡正面照片、商家门店（拍门头）、商家门店（店内）、个人手持身份证照片/手持码牌二选一（共计六张照片）

第二、个人无营业执照且有固定门店（小微商户）
需要准备材料：个人身份证（正反面）、银行卡正面、商家门店（拍门头）、商家门店（店内）、个人手持照片（共计六张照片）

第三、企业有营业执照无固定门店（普通商户）
需要准备：执个人身份证（正反面照片）、银行卡正面照片、商家门店（拍门头）、商家门店（店内）、法人手持身份证照片（共计七张照片）

第四企业有营业执照有固定门店（普通商户）
需要准备材料：营业执照、个人身份证（正反面照片）、银行卡正面照片、商家门店（拍门头）、商家门店（店内）、法人手持照片（共计七张照片）

2.2. 提交材料开通

先下载云闪付收银台，安装后进行进件。注意云闪付收银台没有 iOS 端，只有安卓端。

好了下面是进件流程(一定要仔细看，别出错后被驳回)：

  1. 主要注意写常用手机号与正确的邀请码，错误的邀请码会导致你失去代理商，后面的步骤无法处理。邀请码在详细步骤中
  2. 小微商户的商户名称格式是 商户_姓名 ，普通商户的商户名称格式是 营业执照全称，不要乱写，不要创意，名字错了微信就没办法支付，你的码就浪费了！
  3. 商户的简称，就看着写吧
  4. 上传文件的位置，与都上传什么文件，仔细看，小微商户不需要传营业执照，普通商户多一个营业执照
  5. 等待商户收取短信，短信的发送时间是：上午9:00-11:30 下午：14:00-17:00
  6. 商户收到短信后，就可以进行商户的微信和支付宝认证了，很重要，不认证银联码不能微信支付宝收款，切记！

2.2.1 提交材料详细步骤

下载云闪付收银台，安装，点注册

同意协议 & 填写负责人资料
记得在下方邀请码填机构的邀请码(云闪付收银台邀请码：401293096)。

填写商户信息

  • 无营业执照的商户请选择小微商户（下图左），有营业执照的商户请选择普通商户（下图右）；
  • 小微商户的商户名称必须填写为商户_姓名，否则无法通过审核和微信认证，比如商户_张三；
  • 普通商户的商户名称必须填写为营业执照名称；
  • 商户简称请填写实际店铺门头名称（客户付款成功后会在账单中显示）；
  • 详细地址填写为实际收款常用地址；
  • 客服电话填写负责人的常用手机号，售后专用；
  • 备注可不用填写。

行业一般根据实际情况选择，若是软件等线上行业，这里先选个线下的就行（如百货，商超之类就行），不然后面费率就不对了。
若选错了就会出现下面这种情况，开不了0.38哦.

填写银行卡账户类型

  • 普通商户需要对私入账，请选择个人账户，即法人对私银行卡；
  • 普通商户需要对公入账，请选择企业账户，即企业对公账户；
  • 如果查询不到开户支行名称，请选择所在地市的分行。

上传证件照片

  • 小微商户需要上传资料：身份证+银行卡+门头照片(或摊位)+手拿保证书照片（上传到协议一）。
  • 普通商户需要上传资料：营业执照+身份证+银行卡+门头照片+店铺内景照片（上传到协议二）+店铺收银台照片（上传到协议三）。
  • 保证书下载地址：https://wwk.lanzoum.com/iGIsB0f8562h

注意：店铺门头、内景、收银台照片要体现出是同一家店铺，内景照片要体现出店铺的真实经营内容。

照片按这个填：

商户信息审核

提交资料完成之后，请耐心等待审核，工作日一般 30 分钟内审核完毕，审核成功之后，会收到中国银联系统发出的商户账号和初始密码。

2.3. 二次认证 & 拿到密钥

审核通过后，加服务商进行二次认证。
注意：支付宝认证需要用实体营业执照，微信支付可以用电子版的营业执照截图。

二次认证后需要确认是否拿到后台密钥：

  1. 通过营业执照开通的，后台地址和密钥会直接发送至邮箱中
  2. 没有营业执照的，请联系服务商获取后台密钥。

3. 搭建易支付平台，对接云闪付收银台上的微信和支付宝渠道

3.1. 易支付搭建

3.1.1 源码获取：
微信联系： limo-on 私发

3.1.2 服务器环境

推荐使用宝塔面板一键部署服务器环境。

PHP版本：>=7.1，推荐7.4或8.0
MySQL版本：5.6或5.7

3.1.2 伪静态配置

直接上传后访问即可完成安装！
创建好网站之后，需要配置伪静态才能正常发起支付。以下分别是Nginx、Apache、IIS服务器的伪静态配置方法：

Nginx
如果是nginx，伪静态规则在源码包根目录的nginx.txt文件里面。
将nginx.txt里面的全部内容复制，粘贴到宝塔伪静态配置的输入框里面，点保存即可。

Apache
如果是Apache，伪静态规则在.htaccess里面，默认配置好的，不需要单独修改。

IIS
如果是IIS，伪静态规则在IIS.txt里面，复制粘贴到宝塔伪静态配置的输入框里面，点保存即可。

3.2 易支付支付通道配置

通过审核后会有秘钥发到你邮箱里，直接把秘钥和账号配置到易支付银联里面就可以了

4. 独角卡对接易支付

  1. 在易支付中创建一个普通账号，记下这个账号的 id 和 密钥
  2. 易支付请求网址是网址后面加/submit.php，不然请求没有作用！例如：http://xxx.com/submit.php
  3. 独角卡后台修改易支付微信/易支付支付宝通道的参数：商户id 填第一步的 id，商户key 填易支付请求网址，商户密钥 填第一步的 密钥。
  4. 注意独角卡后台的易支付微信/易支付支付宝通道，支付场景 选择通用，支付方式选择扫码。
  5. 最后启用收款通道，测试收款是否发货并在云闪付后台中核单金额即可。
